# Курсовая работа

## Содержание: 
1. Абстрактный класс **RequestAPI**
2. Класс API **Aircrafts**
3. Абстрактный класс, определяющий методы по работе с файлом, который содержит информацию о самолете **FileWork**
4. Класс для сохранения информации о самолетах **SaveInfoAircraft**
5. Класс, который содержит информацию по самолетам **AircraftInfo**
6. Функция сортировки самолетов по высоте
7. Функция фильтрации самолетов по стране регистрации
8. Функция фильтрации самолетов по диапазону высот


## Установка:
1. Клонируйте репозиторий:
(`git clone`) [GitHub] (https://github.com/zelenyzar/Flightradar2026)
2. Установите зависимости
(`pip requirements.txt`)

## Использование проекта:
1. Откройте проект;
2. Введите данные запрашиваемые программой.
